# Augmented Reality Furniture and Electronics Catalog

## Deskripsi
AR Furniture & Electronics Visualization merupakan aplikasi Augmented Reality (AR) yang dikembangkan menggunakan Unity dan Vuforia. Aplikasi ini memungkinkan pengguna untuk memindai image target dan menampilkan model 3D furniture maupun perangkat elektronik secara real-time pada lingkungan nyata.
Proyek ini dibuat sebagai tugas mata kuliah Game Development untuk mempelajari implementasi teknologi Augmented Reality pada perangkat mobile.

## Tujuan
- Mempelajari konsep Augmented Reality menggunakan Unity dan Vuforia.
- Mengimplementasikan image target tracking.
- Menampilkan objek 3D secara interaktif.
- Memberikan pengalaman visualisasi produk melalui teknologi AR.

  ## Fitur Utama
- Deteksi Image Target menggunakan Vuforia.
- Menampilkan model 3D furniture dan elektronik.
- Tracking objek secara real-time.
- Interaksi pengguna dengan objek AR.
- Tampilan antarmuka yang sederhana dan mudah digunakan.

## Cara Menjalankan
1. Clone repository ini.
2. Buka project menggunakan Unity Hub.
3. Pastikan package Vuforia sudah terinstall.
4. Buka scene utama pada folder Scenes.
5. Jalankan project melalui Unity Editor atau build ke Android.

## Cara Penggunaan
1. Buka aplikasi pada perangkat Android.
2. Arahkan kamera ke image target yang telah disediakan.
3. Sistem akan mendeteksi marker secara otomatis.
4. Model 3D akan muncul di atas image target.
5. Pengguna dapat mengamati objek dari berbagai sudut pandang.
